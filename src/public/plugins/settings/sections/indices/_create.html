索引(Index)配置页

使用Moon-Kibana你必须至少配置一个索引(Index),它用来让ES运行搜索和分析.也用于配置字段.

您可以使用动态索引名 * 作为通配符.如: logstash-*

您可以使用动态索引名,在索引名中的静态文本是用括号表示,比如:[logstash-]YYYY.MM.DD, 注意,这里的每星期是从星期一开始的 — 查阅日期格式文档(English)

Note: 您使用的是按星期查阅的指数(我勒个去,咋翻译啊). Moon-Kibana要求星期在创建索引时使用.(绕口啊) I noticed you're using weekly indices. Kibana requires ISO weeks be used in your index creation. 查阅维基百科有关于ISO标准星期的相关信息(English)Wikipedia: ISO Week Date
